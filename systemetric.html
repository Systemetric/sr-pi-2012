bearing Class¶

class systemetric.Bearing(degrees=None, radians=None)¶

Bases: object

Stores an angle between -180 and 180 degrees. Wraps numbers around such that an angle can never fall outside of these bounds

>>> Bearing(90)
90.0°
>>> Bearing(90) - Bearing(30)
60.0°
>>> Bearing(-120) - Bearing(120)
120.0°
>>> Bearing(60) - Bearing(180)
-120.0°

class systemetric.twowheeledrobot.SingleThreadAccess¶

Bases: object

Allow only one thread access to the resource at a time. At any time a thread can grab control, causing attribute access in other threads to throw an exception, revoking control from the others. Base class members can still be accessed through super.

hasControl¶

Check if the current thread has control - not required in user code. Daemon threads, such as the PID, always have access

takeControl(kick=True)¶

Take control of the object, revoking access to all other threads
